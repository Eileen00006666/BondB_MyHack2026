# BondB
**Build with AI 2026 (MyHack 2026)**

AI-powered relationship intelligence and networking platform built using AI and Google Cloud technologies.


**Overview**

This hackathon helps users analyze relationships, compatibility, and collaboration opportunities between individuals and organizations using AI-driven insights and smart matching.



**Features**

-AI Relationship Analysis

-Smart Matching & Compatibility Scoring

-Data Ingestion & Processing

-Interactive React Frontend

-Google Cloud & Vertex AI Integration

**Installation**

**Clone Repository**

git clone https://github.com/Eileen00006666/BondB_MyHack2026.git

cd BondB



**Backend Setup**

cd backend

npm install

node server.js



**Frontend Setup**

cd frontend

npm install

npm run dev



**Environment Variables**

Create .env inside backend:

**VERTEX_PROJECT_ID = quantum-yen-496509-g3**

**VERTEX_LOCATION = us-central1**
